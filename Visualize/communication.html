Labels

Add labels with labs() function:

ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy)
) + 
  geom_point(
    mapping = aes(color = class)
  ) +
  geom_smooth(se = FALSE) +
  labs(
    x = "Engine displacement (L)",
    y = "Highway fuel economy (mpg)",
    color = "Car type",
    title = "Fuel efficiency generally decreases with engine size",
    subtitle = "Two seaters (sports cars) are an exception because of their light weight",
    caption = "Data from fueleconomy.gov"
  )
`geom_smooth()` using method = 'loess' and formula = 'y ~ x'
Tip

Avoid titles that just describe what the plot is, e.g., “A scatterplot of engine displacement vs. fuel economy”. caption often used to describe source of data.

Also can use mathematical equations instead of text strings by replacing "" with quote (see also ?plotmath).

df <- tibble(
  x = 1:10,
  y = cumsum(x^2)
)

ggplot(
  data = df,
  mapping = aes(x = x, y = y)
) +
  geom_point() +
  labs(
    x = quote(x[i]),
    y = quote(sum(x[i]^2, i == 1, n))
  )

Annotations

Often useful to label individual observations or groups of observations.

\(\rightarrow\) geom_text() is similar to geom_point(), BUT has additional aesthetic: label.

# First pull out cars with highest engine size per drive type
label_info <- mpg |>
  group_by(drv) |>
  arrange(desc(displ)) |>
  slice_head(n = 1) |>
  mutate(
    drive_type = case_when(
      drv == "f" ~ "front-wheel drive",
      drv == "r" ~ "rear-wheel drive",
      drv == "4" ~ "4-wheel drive"
    )
  ) |>
  select(displ, hwy, drv, drive_type)

Tip

There are many more geoms available in ggplot2 for annotations, besides geom_text() and geom_label():

  • geom_hline() and geom_vline() adds reference lines. (Usually linewidth = 2 and color = "white" to not draw attention away from data)

  • geom_rect() draws rectangles around points of interest. Boundaries defined by xmin,xmax,ymin,ymax.

  • geom_segment() with arrow arguments drwas attention to a point with an arrow. Aesthetics x, y, xend, yendto define start and end locations.

Another function for adding annotations is annotate()

Tip

Rule of thumb:

\(\rightarrow\) Use geoms when highlighting subset of data.

\(\rightarrow\) Use annotate() when adding one of few annotation elements to a plot

trend_text <- "Larger engine sizes tend to have lower fuel economy." |>

Scales

Scales control how aesthetic mappings will be displayed

Default scales

ggplot2 automatically adds scales:

ggplot(
  data = mpg, 
  mapping = aes(x = displ, y = hwy)
) +
  geom_point(
    mapping = aes(color = class)
  )
# Behind the scenes
ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy)
) + 
  geom_point(
    mapping = aes(color = class)
  ) +
  scale_x_continuous() + # puts values of displ on continuous line on x-axis
  scale_y_continuous() +
  scale_color_discrete() # chooses colors for each of the class of car

Scales are named according to type of variable:

  • continuous
  • discrete
  • datetime
  • date

Reasons to override defaults:

  • changing paramters of default scale, allowing e.g. to change breaks on axes, or key labels on the legend
  • replacing scales, and using different algorithm

Axis ticks and legend keys

Axis and legends are calles guides. Axis are used for x/y aesthetics; legends for everything else.

Two arguments affecting appearance of ticks on axis and keys on legend: breaks and labels:

  • break: controls position of ticks; - values associated with keys
  • labels: controls text labels
ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy, color = drv)
) + 
  geom_point() +
  scale_y_continuous(breaks = seq(from = 15, to = 40, by = 5))

Can use labels same way, or set NULL to suppress. For discrete scales for categorical variables, labels can be named list of existing level names and desired labels for them:

# Example: Rescaling and relabeling x-axis
ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy)
) + 
  geom_point() +
  scale_x_continuous(
    breaks = seq(from = 2, to = 7, length.out = 3),
    labels = c("A", "B", "C"))
# Example: labels = NULL
ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy, color = drv)
) + 
  geom_point() +
  scale_x_continuous(labels = NULL) +
  scale_y_continuous(labels = NULL) +
  scale_color_discrete(labels = c(
    "4" = "4-wheel",
    "f" = "front",
    "r" = "rear"
  ))

Legend layout

Controls overall position of the legend using theme() setting - they control non-data parts of the plot. Theme setting legend.position controls where the legend is drawn:

base <- ggplot(
  data = mpg, 
  mapping = aes(x = displ, y = hwy)
) + 
  geom_point(
    mapping = aes(color = class)
  )

base + theme(legend.position = "right") # default
base + theme(legend.position = "left")
base +
  theme(legend.position = "top") +
  guides(color = guide_legend(nrow = 3))
base +
  theme(legend.position = "bottom") +
  guides(color = guide_legend(nrow = 3))
Note

guides() along with guides_legend() or guide_colorbar() controls the display of individual legends. nrowcontrols number of rows the legend uses; override.aes overrides one of the aesthetics, e.g., making points bigger. Useful when used a low alphato display many points on a plot.

ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy)
) + 
  geom_point(
    mapping = aes(color = class)
  ) + 
  geom_smooth(se = FALSE) +
  theme(legend.position = "bottom") +
  guides(color = guide_legend(nrow = 3, override.aes = list(size= 4)))
`geom_smooth()` using method = 'loess' and formula = 'y ~ x'
Note

Argument in guides() matches name of aesthetic, just like in labs().

Replacing a scale

Two types of scales likely to switch out: continuous position scales and color scales.

# Example - log transforming to better see relationship
# Left
ggplot(
  data = diamonds,
  mapping = aes(x = carat, y = price)
) + 
  geom_bin2d()
`stat_bin2d()` using `bins = 30`. Pick better value `binwidth`.
# Right
# Note: scales are labelled with transformed values
ggplot(
  data = diamonds,
  mapping = aes(x = log10(carat), y = log10(price))
) + 
  geom_bin2d()
`stat_bin2d()` using `bins = 30`. Pick better value `binwidth`.
# Instead doing transformation in aesthetic mapping, transform with scales
# Note: Scales are labelled with original values
ggplot(
  data = diamonds,
  mapping = aes(x = carat, y = price)
) + 
  geom_bin2d() +
  scale_x_log10() +
  scale_y_log10()
`stat_bin2d()` using `bins = 30`. Pick better value `binwidth`.

Another scale frequently customized is color. Default categorical scale picks colors that are evenly spaced around the color wheel.

Useful alternative: ColorBrewer - tuned colors for color blindness.

ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy)
) + 
  geom_point(
    mapping = aes(color = drv)
  )
# For people with red-green color blindness: 
ggplot(
  data = mpg,
  mapping = aes(x = displ, y = hwy)
) +
  geom_point(
    mapping = aes(color = drv, shape = drv)
  ) +
  scale_color_brewer(palette = "Set1") # See documentation for palettes
Tip

For ColorBrewer scales see online documentation at https://colorbrewer2.org/.

Tip

For continuous color, use scale_color_gradient() or scale_fill_gradient(). For diverging scale use scale_color_gradient2() - allows to give negative and positive values different color.

Zooming

Three ways to control plot limits:. 1. Adjust what data are plotted. 2. Set limits in each scale. 3. Set xlim and ylim in coord_cartesian()

# Left
ggplot(
  data = mpg, 
  mapping = aes(x = displ, y = hwy)
) +
  geom_point(
    mapping = aes(color = drv)
  ) +
  geom_smooth()
`geom_smooth()` using method = 'loess' and formula = 'y ~ x'
# Right
mpg |> 
  filter(displ >= 5 & displ <= 6 & hwy >= 10 & hwy <= 25) |>

    mapping = aes(x = displ, y = hwy)
  ) +
  geom_point(
    mapping = aes(color = drv)
  ) + 
  geom_smooth() +

Themes

Customizing non-data elements of the plot with a theme:

ggplot(
  data = mpg, 
  mapping = aes(x = displ, y = hwy)
) +
  geom_point(
    mapping = aes(color = drv)
  ) +
  geom_smooth(se = FALSE) +
  theme_bw()
`geom_smooth()` using method = 'loess' and formula = 'y ~ x'
Note

ggplot includes 12 themes: *_bw(), *_light(), *_classic(), *_linedraw(), *_dark(), *_minimal(), *_gray(), *_void(); Many more in add-on packages like ggthemes.

ggplot(
  data = mpg, 
  mapping = aes(x= displ, y = hwy, color = drv)
) +
  geom_point() +
  labs(
    title = "Larger engine sizes tend to have lower fuel economy",
    caption = "Source: https://fueleconomy.gov."
  ) +
  theme(
    legend.position = c(0.6, 0.7), # position of legend in plot
    legend.direction = "horizontal", # layout of items in legend
    legend.box.background = element_rect(color = "red"),
    plot.title = element_text(face = "bold", hjust = 1),
    plot.title.position = "plot",
    plot.caption.position = "plot",
    plot.caption = element_text(hjust = 0),
    axis.title = element_text(face = "bold", color = "blue")
  )
